# 《无名之界》

一个由 LLM 驱动的命令行神明模拟器。你扮演一个正在注视世界的存在，从石器时代开始，见证文明的生长与消亡。

---

## 世界观

这个世界建立在两种根本底质之上：**凝质（Sang）**，构成一切有形之物；**息（Veth）**，渗透万物的意识场域。两者并非对立，而是同一实在的两面。科技与魔法在这里不是分类，而是同一力量的不同读取方式。

世界从石器时代起步。没有可参考的历史，没有已知的神话。一切都在发生之中。

---

## 特性

- **每个命名居民都有完整的个人时间线** — 出生、相遇、变异、祈祷、死亡，全部记录
- **真实流逝的世界时间** — 后台线程持续推进，每 60 秒真实时间 = 世界 1 年；离线期间时间也在流逝，回来时会自动补偿
- **三级变异系统** — 表层/功能/本质变异，可扩散感染周围的人、物、地形；玩家行为会偏移变异方向
- **NPC 自主权** — 居民有自己的信仰、祈祷、记忆碎片，以及向神明提出改变世界的提案
- **自生长代码** — 当游戏检测到新兴系统（贸易、宗教、战争……），LLM 会自动生成对应的 Python 模块并热加载进运行中的游戏；模块崩溃时自动修复，修复成功后居民会重新通知你
- **世界神话库** — 重大事件自动沉淀为神话，流传于居民口中
- **别的神明** — 你不是唯一的存在

## 安装

```bash
pip install openai python-dotenv
```

复制 `.env.example` 为 `.env`，填入你的 [OpenRouter](https://openrouter.ai) API Key：

```
OPENROUTER_API_KEY=sk-or-v1-你的key
```

---

## 启动

```bash
python main.py
```

首次运行自动创世。之后每次启动会读取上次的存档和时间戳，世界继续。

---

## 指令

| 指令 | 说明 |
|------|------|
| `赐予 <物品>` | 消耗 10 信仰，向部落赐予一件物品 |
| `施放 <奇迹>` | 消耗 50 信仰，降下神迹 |
| `回应 <名字> <类型>` | 回应祈祷，类型：答应 / 无视 / 惩戒 / 赐福 |
| `凝视 <名字或地点>` | 深度洞察某人或某地（不消耗回合） |
| `状态` | 当前世界概况 |
| `人群` | 所有命名居民列表 |
| `人 <名字>` | 某人的完整人生事件流 |
| `人物` | 特殊实体 |
| `变异` | 当前活跃变异 |
| `神话` | 世界神话库 |
| `祈祷` | 等待回应的祈祷列表 |
| `模块` | 自生成扩展系统状态 |
| `帮助` | 显示指令列表 |
| `退出` | 保存并离开（世界继续流逝） |

---

## 技术栈

- Python 3.11+
- OpenRouter API（Claude haiku-4-5 日常调用，sonnet-4-5 代码生成）
- `threading` 后台世界模拟线程
- `exec()` 热加载 LLM 生成的模块
